介绍：webpack5+Vue3项目，配合Umi的MFSU技术提高webpack构建打包速度。

## mfsu方案启动方式

```sh
$ yarn install 
$ yarn run dev
```


## mfsu + esbuild方案启动方式

```sh
$ yarn install 
$ yarn run dev:esbuild
```
